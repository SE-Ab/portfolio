Toonscode Learning App

EdTech, AI-powered

Mobile-first coding learning platform with AI-driven feedback and real-time code execution across multiple programming languages.

Detailed Description

An ambitious project to create a comprehensive mobile learning environment for coders. The app supports multiple languages like Python, Java, and JavaScript, offering interactive lessons, quizzes, and a real-time code editor. The core feature is an AI-powered mentor that provides instant feedback on user's code, identifies errors, suggests improvements, and explains complex concepts, adapting to the user's learning pace.

Challenges

Integrating a secure and performant real-time code execution sandbox for multiple languages on a mobile platform was a major hurdle. Another challenge was developing an AI model that could provide meaningful, context-aware feedback without being computationally expensive for the user's device.

Solutions

We utilized lightweight, containerized environments in the backend, accessed via a secure API, to handle code execution. For the AI mentor, we fine-tuned a smaller language model specifically for code analysis and leveraged on-device processing for simpler checks, only sending more complex queries to the backend, optimizing both performance and cost.

Smart Raseed Mobile App

Spring Boot & React Native

Receipt management solution for small businesses with quick creation, encrypted storage, and analytics dashboard.

Detailed Description

Smart Raseed is a digital receipt management tool designed to help small business owners move away from paper-based systems. Users can generate, send, and store digital receipts instantly. The app features a powerful dashboard for tracking sales, analyzing revenue trends, and managing customer data. All data is stored securely with end-to-end encryption.

Challenges

Ensuring data integrity and security for sensitive financial information was paramount. The app also needed to be highly performant and functional in areas with limited internet connectivity, which is common for many small businesses.

Solutions

Implemented JWT-based authentication and encrypted data storage using Spring Security on the backend. For offline capabilities, we used React Native's AsyncStorage to cache data locally and implemented a background sync mechanism that updates the central database once a stable connection is available.

Truly Mentor Academy

Full-Stack Education Platform

Complete education management system with role-based dashboards, fee tracking, attendance, and online testing modules.

Detailed Description

This is an all-in-one management platform for educational institutions. It provides distinct, role-based dashboards for administrators, teachers, students, and parents. Key features include automated fee management and payment reminders, digital attendance tracking, an online examination module with automatic grading, and a communication portal for seamless interaction between stakeholders.

Challenges

The main challenge was designing a flexible, multi-tenant architecture that could cater to the unique needs of different institutions while maintaining a single, scalable codebase. Handling complex permission structures and ensuring data privacy across different user roles was also a significant task.

Solutions

We built the platform on a microservices architecture using Spring Boot, allowing for independent scaling and development of different modules (e.g., fees, exams). A robust role-based access control (RBAC) system was implemented from the ground up to manage permissions granularly, ensuring users could only access data relevant to their role.
